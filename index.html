About Internet

Interent is the way we use computers to communicate with others.

Firstly, we go to a web page which computer there could send a http request to a server,
and then server sends appropriate HTML documents back to user's computer and its browser
does interpretation and presents it on the screen.



Language we use here

HTML - HyperText Markup Language.
It's about the format of documents of the primary contents on the web. Knowing how this language has been used can help us to know the web more.

Tags, Elments, Attribute

They are basic vocabularies of HTML.
Tags present inline or block text, some of them are paired which include opening and closing, such as bold, emphasis, paragraph; while others only have opening without closing part.

Elements are everything within the a set of opening and closing tags. They are either inline or block. Block elements are invisible box around the contents inside of them.

Attribute is about a property of a HTML element; for example, we use href of an anchor tag to link a url.
